# systemslibrarian.dev

Personal developer site for **Paul Clark** — Application Systems Analyst, MLIS, and builder of things that sit at the intersection of library science, security engineering, and faith-driven development.

---

## About

By day I manage library systems and internal tooling for [Leon County Public Library](https://leoncountylibrary.org) in Tallahassee, FL — spanning SirsiDynix, ASP.NET Core applications, WCAG accessibility compliance, and physical access control monitoring.

By night I build cryptographic demos, Scripture tools, and the occasional air-gapped QR-code file transfer utility.

My GitHub handle is `systemslibrarian` because the two halves of that word are not in conflict.
